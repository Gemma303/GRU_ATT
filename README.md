# GRU_ATT

Implementation of the paper: A Compact Neural Network for Predicting Cross-Sectional Stock Returns from Daily Price History

This paper presents a compact model for forecasting the cross-section of future stock returns using historical return data. The model employs Gated Recurrent Units (GRUs) to capture the temporal pattern of each stock’s own return series and an attention mechanism to capture the spatial correlations between different stocks.
